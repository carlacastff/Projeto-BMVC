# BMVC-oo
Disco 2000 — Projeto BMVC

**Disco 2000** é um projeto desenvolvido para o **BMVC**, com o objetivo de servir um site com **(HTML, CSS, JavaScript, Python e SQLite)** através de um **servidor Python usando o framework Bottle**.

A proposta é criar um site com propósito claro, conteúdo legível e arquivos CSS e JS plenamente funcionais e ativos.

---

## Visão Geral do Projeto

O **Disco 2000** é um website que simula uma **loja de discos de vinil retrô**.

O site apresenta:
- Um **menu de navegação** com links e ícones funcionais;
- Uma **seção de destaque** com barra de pesquisa estilizada;
- Uma área de **“Mais Vendidos”**, mostrando produtos em destaque com efeitos visuais;
- Uma **seção Premium**, promovendo um plano de assinatura de vinis raros;
- Um **carrossel de gêneros musicais**, controlado via JavaScript;
- Um **rodapé estilizado** com informações fictícias da marca;
- Um **banco de dados** funcional.

## Sistema de Gestão (CRUD Completo)
- **Criação**: Cadastro de produtos com upload de imagens;
- **Leitura**: Pesquisa e visualização de produtos;
- **Atualização**: Edição completa de informações e mídias;
- **Exclusão**: Remoção segura com confirmação.
